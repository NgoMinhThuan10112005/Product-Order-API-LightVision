Product and Order API (1.0.0)

Download OpenAPI specification:

API Support: support@lightvision.com License: Apache 2.0

A REST API for managing products and orders with JWT authentication. This system provides secure endpoints for product catalog management, user authentication, and order processing with automatic transaction handling.

Authentication

User registration and login management with JWT token generation

Register a new user

Creates a new user account with encrypted password using BCrypt hashing algorithm

Request Body schema: application/json
required

User registration details

username
required
string [ 3 .. 50 ] characters

Unique username for the account

password
required
string <password> >= 6 characters

Password (will be encrypted with BCrypt)

email
required
string <email>

User's email address

role
string
Default: "USER"
Enum: "USER" "ADMIN"

User role in the system

Responses

Local Development Server

http://localhost:8080/api/auth/register

Request samples

  • Payload
Content type
application/json
{
  • "username": "johndoe",
  • "password": "SecurePass123!",
  • "email": "johndoe@example.com",
  • "role": "USER"
}

Response samples

  • 200
  • 400
Content type
application/json
{
  • "id": 1,
  • "username": "johndoe",
  • "email": "johndoe@example.com",
  • "role": "USER"
}

Order Management

Order placement and transaction processing with automatic rollback

Place a new order

Creates an order for a user with automatic inventory management. Features transaction rollback if any item is out of stock, ensuring data consistency. Requires authentication.

Authorizations:
bearerAuth
Request Body schema: application/json
required

Order details including user ID and list of items

userId
required
integer <int64>

ID of the user placing the order

required
Array of objects (OrderItemRequest) non-empty

List of products and quantities to order

Responses

Local Development Server

http://localhost:8080/api/orders
